| Kolom                     | Deskripsi                                                                                          |
| :------------------------ | :------------------------------------------------------------------------------------------------- |
| `user_id`                 | ID unik untuk setiap pengguna.                                                                     |
| `display_name`            | Nama tampilan (username) pengguna.                                                                 |
| `name`                    | Nama lengkap pengguna.                                                                             |
| `email`                   | Alamat email pengguna.                                                                             |
| `phone`                   | Nomor telepon pengguna.                                                                            |
| `total_tracking_events`   | Total jumlah interaksi (melihat, membuka, dll.) pengguna dengan tutorial.                          |
| `total_completed_modules` | Total jumlah modul yang telah diselesaikan oleh pengguna.                                          |
| `total_submissions`       | Total jumlah pengiriman kuis oleh pengguna.                                                        |
| `avg_submission_rating`   | Rata-rata nilai (rating) yang diterima untuk pengiriman kuis oleh pengguna.                        |
| `avg_study_duration`      | Rata-rata durasi belajar untuk perjalanan (journey) yang diselesaikan oleh pengguna (dalam menit). |
| `avg_completion_rating`   | Rata-rata nilai pengiriman dari perjalanan yang telah diselesaikan oleh pengguna.                  |
| `avg_exam_score`          | Rata-rata skor yang dicapai pengguna dalam ujian.                                                  |
| `exam_pass_rate`          | Tingkat kelulusan ujian (persentase ujian yang berhasil dilewati) oleh pengguna.                   |
